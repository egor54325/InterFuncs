## Configuration

In the `config` directory, there are configuration files:
- `cookies.json` — stores cookies data.  
  Example:
  ```json
  {
      "session": "your_session_cookie_here"
  }
  ```
- `params.json` — contains additional parameters.  
  Example:
  ```json
  {
      "param1": "value1",
      "param2": "value2"
  }
  ```
